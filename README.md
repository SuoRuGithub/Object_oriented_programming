# Object Oriented Programming

本仓库包含了我在2024年“面向对象程序设计”的课程中完成的作业（不包含大作业）.

本仓库组织结构如下：

```
/
├── README.md
├── hw1
|   ├── .vscode
|   ├── build
|   │   ├── ……
|   │   └── hw1
|   ├── CMakeLists.txt
|   ├── ……
|   └── hw1.cpp
├── hw2
└── ……
```

请按照下面的方式编译并执行代码（以hw1为例）：

```bash
hw1/build$ cmake ..
hw1/build$ make
hw1/build$ ./hw1
```
